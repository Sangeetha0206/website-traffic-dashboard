# Website Traffic Analytics Dashboard

This project analyzes website traffic data and visualizes key metrics using Microsoft Excel.

## Key KPIs
- Audience Reach (Users)
- Engagement Count (Sessions)
- Drop-off Rate (Bounce Rate)
- Engagement Time (Avg Session Duration)

## Features
- Data cleaning and preparation
- Pivot tables and charts
- KPI summary cards
- Interactive dashboard design

## Tools Used
- Microsoft Excel
